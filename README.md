# Dependency

- Python
- pip
- matplotlib

# Setup

```bash
pip install matplotlib
```

# Usage
プロジェクトのルートディレクトリで、以下のコマンドを実行する。種類は(osaka・bunkyo・nagoya)の3種類。

```bash
python main.py [好みのおっぱい]
```
